About

Adgnosco is Jean Liu's capstone project at Hackbright Academy.


Disclaimer: Adgnosco would not be possible without OpenFace. I'd like to thank Brandon Amos and his team for being so generous with OpenFace. I am not making any money off Adgnosco. It is purely an academic project for personal use. Please check them out at http://cmusatyalab.github.io/openface


Adgnosco is currently designed to be a passive security system when partnered with a Nest Camera (subscription required). It is currently a work-in-progress as I have the following features I'd like to add:

    text message or email alert when an unrecognized person is identified in your home
    a user-friendly method to upload pictures of people they'd like the camera to identify
    a UI that is geared towards home usage
    see if I can incorporate one of those door code things into this
